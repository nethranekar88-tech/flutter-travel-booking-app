# Flutter Travel Booking App

A travel booking application built with Flutter.

## Features

- ✈️ Destination explorer
- 📋 Travel details page
- 📝 Booking form
- 📅 Date picker
- 👥 Traveler counter
- 🏨 Room selection
- ✅ Booking confirmation popup

## Tech Used

- **Flutter** - Cross-platform mobile framework
- **Dart** - Programming language
- **Material UI** - Google's design system

## Getting Started

### Prerequisites

- Flutter SDK: [Install Flutter](https://flutter.dev/docs/get-started/install)
- Dart SDK (included with Flutter)
- An IDE like VS Code or Android Studio

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nethranekar-dev/flutter-travel-booking-app.git
cd flutter-travel-booking-app
```

2. Get the dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## Project Structure

```
lib/
  ├── main.dart              # App entry point
  ├── screens/               # App screens
  │   ├── home_page.dart
  │   ├── destination_page.dart
  │   ├── booking_page.dart
  │   └── login_page.dart
  └── widgets/               # Reusable widgets
```

## How to Use

1. Launch the app
2. Browse available destinations
3. Select a destination to view details
4. Fill in booking details (dates, travelers, rooms)
5. Confirm your booking
